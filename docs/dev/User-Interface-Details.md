Part of this project is the creation of a user interface (UI). The main goal of the UI is to allow for the completion of the active learning loop, allowing scientists to annotate training data for machine learning models, view predictions made by the machine learning models and correct predictions to allow for retraining and validation of models.

[Here](https://github.com/ukaea/viz-annotation/tree/main/services/ui) is a link to the main UI code inside the repo.
