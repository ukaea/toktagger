# Building the Docs

We use [MKDocs](https://www.mkdocs.org/) for our documentation. To build and view the docs locally

1. Check you've installed the relevant dependancies:

```sh
uv sync --group dev
```

2. Run the `mkdocs` command

```sh
mkdocs serve
```

3. Navigate to [http://localhost:8000](http://localhost:8000) and view the docs!
